# Calculadora Android (Jetpack Compose)

## 📱 Sobre o projeto

Este é um aplicativo simples de calculadora desenvolvido em Kotlin
usando Jetpack Compose no Android Studio.

Ele foi criado com fins educacionais para praticar: - Interface com
Compose - Manipulação de estado - Lógica básica de operações matemáticas

------------------------------------------------------------------------

## ⚙️ Funcionalidades

-   Soma (+)
-   Subtração (-)
-   Multiplicação (\*)
-   Divisão (/)
-   Botão de limpar (C)
-   Interface inspirada na calculadora do iPhone 🍎

------------------------------------------------------------------------

## 🎨 Interface

-   Fundo escuro
-   Botões arredondados
-   Operações em destaque (laranja)
-   Display grande no topo

------------------------------------------------------------------------

## 🚀 Como rodar o projeto

1.  Abra o projeto no Android Studio
2.  Aguarde o Gradle sincronizar
3.  Clique em Run ▶️
4.  Execute em um emulador ou celular

------------------------------------------------------------------------

## 🛠️ Tecnologias usadas

-   Kotlin
-   Jetpack Compose
-   Material 3
